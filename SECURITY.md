# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities **privately** — do not open a public
issue.

- Email the maintainer listed in `pyproject.toml` (`authors`) with the details.
- Include steps to reproduce and an estimate of the impact.
- You will receive an acknowledgement within a few days.

We will work with you to assess and address the issue, and will credit you in
the release notes unless you prefer to remain anonymous.
